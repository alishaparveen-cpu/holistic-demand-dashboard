Source Reconciliation

Every Screening-Call booking traced to its real source (call-match + recovered UTM), reconciled to the total — so we can see what each channel contributes, the lead→book rate where trackable, and what’s untagged. Weeks: latest 8 complete (Mon-based).
① Lead → Booked by channel (calls: → answered → relevant → booked)
For calls: total inbound → answered / not-answered → relevant (AI says real appointment intent) → of those, booked vs didn’t. For web / Practo: leads → booked / didn’t. Shown only for channels tied to this clinic without needing a booking — GMB call, GMB website, Google paid call, Google paid web (single-clinic cities), Practo. Google-paid-web in multi-clinic cities + Meta + Organic are city-level / un-isolatable, so they appear in ② (booked side) only.
② Bookings by source (deduped per patient/week — reconciles to total)
③ Untagged bookings — new vs repeat
④ Bookings → Done → Purchased → Revenue
Booked = unique-patient Screening Calls (matches the ① total). Done = completed consults; Purchased = completed + paid invoice; Revenue = ₹ collected. B→D% is the real no-show/completion rate.
